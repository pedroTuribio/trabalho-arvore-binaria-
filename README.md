Sistema de Achados e Perdidos - Aeroporto

Projeto final desenvolvido para a matéria de Estrutura de Dados. O sistema gerencia o controle de objetos esquecidos em um aeroporto utilizando uma Árvore Binária de Busca (BST) para garantir eficiência nas operações de cadastro, busca e remoção.
Objetivo do Projeto

Demonstrar a aplicação prática de uma árvore binária onde cada item esquecido (como celular, carteira ou mochila) vira um nó na memória, ordenado automaticamente pelo número do seu protocolo.

 Inserção (add): Organiza os dados na árvore (menores à esquerda, maiores à direita).
Busca (search): Localiza o item pelo número do protocolo com complexidade de tempo O(log n).
 Remoção (remove): Trata a exclusão de nós com zero, um ou dois filhos (substituição pelo sucessor).
 Relatório (in_order): Varre a árvore em ordem simétrica para listar os itens de forma crescente.
